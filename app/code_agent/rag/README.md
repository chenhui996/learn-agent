## 阿里云百炼知识库

### 创建知识库

1. 打开官网，登陆阿里云百炼官网（不是阿里云
2. 选择应用
3. 选择 “数据连接”，也就是应用数据。
4. 应用数据创建完，才可以去知识库创建对应的 知识库，因为创建知识库时，会让你选择 **应用数据**。
5. 知识库创建，基本都默认，其中将 **切片长度** 设置成 2000 即可，先稍微大一点，默认600有点小。
6. 创建 某个 知识库，就成功了。
